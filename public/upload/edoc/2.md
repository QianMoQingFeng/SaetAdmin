## 如何使用变量
### php 控制器
```php
	$name = '阡陌清风';
	$this->assign('name',$name);
```
### html使用方式
```html
	<!-- 通过thinkphp原渲染 -->
	<div>{$name}</div>
```
### javascript使用方式
```javascript
	
	<!-- js使用方式,$data包含全部php assign渲染的变量 -->
	console.log($data);
	console.log($data.name); // 阡陌清风

	
```
### vue使用方式 [推荐] <font color="red">注意下划线</font>
```javascript
	<template>
	    <div>
			{{name_}}
	    </div> 
	</template>
	
	<script>
	const App = {
		created(){
			alert(this.name_)
		}
	}
```
